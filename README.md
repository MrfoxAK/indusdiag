# IndusDiag — Industrial AI Troubleshooting Agent

IndusDiag is an AI-powered industrial troubleshooting agent designed to analyze sensor logs and operational telemetry from industrial systems.

The agent detects anomalies such as:

- Sensor spikes
- Frozen sensors
- Sensor drift
- Missing data
- Out-of-range values

After detecting anomalies, IndusDiag generates a structured troubleshooting report including:

- Issue detected
- Evidence
- Possible root causes
- Severity
- Recommended actions
- Confidence level

The system combines deterministic anomaly detection with LLM-based reasoning.

## Example Use Cases

- Industrial IoT sensor monitoring
- Conveyor belt diagnostics
- Furnace monitoring
- Gateway data validation
- Process anomaly detection
